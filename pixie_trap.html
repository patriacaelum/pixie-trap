Saves the current canvas as PXT to disk.

onLeftDown(event)

Records the location of the mouse click.

If the Move tool is selected, then a hitbox is selected. If the Draw tool is selected, then a hitbox is drawn.

onLeftUp(event)

Records the size of the drawn rectangle and renders it.

onMiddleDown(event)

Records the location of the mouse click.

onMotion(event)

The action of motion depends on the button that is held down.

If the left button is held down and the Move tool is selected, then the hitbox is moved. If the left button is held down and the Draw tool is selected, then the hitbox is drawn. If the middle button is held down, the canvas is panned.
